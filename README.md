# API Documentation

- Endpoint 1
URL: /api/endpoint1
Method: GET
Description: Retrieve data related to endpoint 1.
Parameters:
param1: Description of parameter 1 (e.g., String)
Response Format:
json
Copy code
{
  "key": "value"
}


- Endpoint 2
URL: /api/endpoint2
Method: POST
Description: Submit data to endpoint 2.
Parameters:
param2: Description of parameter 2 (e.g., Object)
Response Format:
json
Copy code
{
  "success": true
}
